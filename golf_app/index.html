This site enable users to calculate their World Handicap System (WHS) handicap based on an initial WHS handicap and latest round Stableford results.

It is intended to be used by golfers who would like to keep track of their "real" handicap but who play lots of non-handicap counting (e.g. social) golf .

The site can be used in one of two ways:

Method 1:

First time users will need to upload 2 spreadsheets, a score spreadsheet and a course spreadsheet. The score spreadsheet must have at least 3 official handicap scores which act as a starting point. Registered users who have already uploaded these file need to simply log in and enter their latest round's score. A new virtual handicap will be calculated and stored for next use. Sample files are available for download. The uploaded files must be in the same format as these sample files.

Move the mouse over the arrow icon to open the dropdown menu. Note that these registration and log-in pages are still under construction so currently, the only option is Method 2 😒

Method 2:

This method enables users to run a web-based script. This script prompts the user to upload two spreadsheets, one containing their scores (comp and non-comp) and one containing course details of all courses and tees they have included in their score sheet. There are sample spreadsheets available for download below and user's uploaded files must be in this format. Also, since the WHS handicap is calculated after a round and this determines the daily handicap for the next round, the order of rounds is important. The more rounds that are included in the score file, the more accurate the estimation will be of their virtual handicap.

Download Sample Score File

This file contains two sheets, one named Score and the second named Artificial Rounds. The Score sheet contains the per-hole stroke results for each hole and round

Additional information is also included here, such as the initial GA handicap (which must be in the correct cell, E2) and the round's PCC or Playing Conditions Calculation.

Remember that the GA handicap is calculated from your current round but it only applies from your next round. So the initial GA handicap value should be the one applying at the start of that round (generated by previous round!)

For Par type rounds, simply enter the final score in the appropriate cell. You may enter the individual hole stroke if you wish but it is not necessary to calculate your points

The Artificial Rounds sheet records the relevant data from your initial 3 GA artifically generated rounds used to initialize the calculations. If you have more than 20 rounds, you don'd need to change this

For calculation purposes, only the last 20 rounds are required since the algorithm works on an "8 of 20 scores" system.

Strictly speaking, all scores from the last 12 months are required to take the Soft and Hard Caps into account

Hopefully, it will all make perfect sense when you see the files 🤞

Download Sample Course File

This file also contains two sheets, one named Ratings and the second named Par_Index. The Ratings sheet contains the Slope and Scratch Ratings for each course and tees that is inlcuded in the Scores sheet.

The Par_Index sheet contains the par and associated index for each hole for all the courses and tees that are contained in the Score file

The app will return an error if any of the courses or tees that are included in the Score files are not included in the Courses file 😞
