You can run a web-based Battery SoC app by clicking the button below. Use the same file as for the main app as input by clicking on Browse Files

The app runs as soon as the file is uploaded and uses the default values shown. The default start and finish dates are one month before the last entry in the file and the date of the last entry respectively

You can enter your own parameters and start and finish dates while the app is running by simply placing the cursor in the appropriate date box and eitehr entering new data or by selecting from the drop down calendar.

If you encounter a date error, check that the dates are valid (e.g. is the end date later than the start date?). If they are invalid, the app will accept new, valid dates while still running.

Also note that, unlike the executable chart outputs, the web-based ones are not zoomable. Therefore, if the date range is too long, the chart will be difficult to read 😞
